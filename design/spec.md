# Design System Specification

## Colors
| Token | Hex | Tailwind Utility |
|---|---|---|
| Primary | #059669 | `bg-primary`, `text-primary` |
| Secondary | #3b82f6 | `bg-secondary`, `text-secondary` |
| Success | #10b981 | `bg-success`, `text-success` |
| Error | #ef4444 | `bg-error`, `text-error` |
| Background | #ffffff | `bg-background` |
| Text | #1f2937 | `text-text` |
| Muted | #9ca3af | `text-muted` |

## Typography
- **Font Family**: Open Sans
- **Headings**: Bold, `text-2xl`, `text-3xl`
- **Body**: Regular, `text-base`
- **Labels**: SemiBold, `text-sm`

## Spacing
- Base unit: 4px (Tailwind standard)
- Container Padding: `p-6` (24px)
- Gap: `gap-4` (16px)

## Components
- **Button**: Height 48px, Rounded 8px (`rounded-lg`)
- **Card**: White background, Shadow (`shadow-sm`), Rounded 12px (`rounded-xl`)
- **Input**: Height 48px, Border Gray-300, Rounded 8px
